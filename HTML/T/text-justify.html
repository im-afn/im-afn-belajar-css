text-justify

text-justify dalam CSS digunakan untuk menentukan bagaimana teks yang diratakan (justified text) harus diatur dalam sebuah elemen. Properti ini berfungsi untuk mengontrol distribusi ruang tambahan yang ditambahkan antara kata-kata ketika teks diratakan.

Text Justify: Auto

Teks ini menggunakan metode default browser untuk meratakan teks. Teks ini akan menambahkan ruang di antara kata-kata secara otomatis.

Text Justify: None

Teks ini tidak memiliki penyesuaian khusus untuk meratakan teks. Ruang tambahan hanya ditempatkan di antara kata-kata.

Text Justify: Inter-Word

Teks ini menambahkan ruang tambahan di antara kata-kata untuk meratakan teks. Metode ini biasanya menghasilkan tampilan yang lebih rapi dan teratur.

Text Justify: Inter-Character

Teks ini menambahkan ruang tambahan di antara karakter untuk meratakan teks. Metode ini dapat berguna untuk bahasa yang memiliki karakteristik huruf yang kompleks.
