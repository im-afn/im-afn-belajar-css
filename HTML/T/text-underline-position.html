text-underline-position

text-underline-position dalam CSS digunakan untuk mengontrol posisi garis bawah (underline) relatif terhadap teks. Properti ini memungkinkan Anda menyesuaikan posisi garis bawah untuk mendapatkan efek visual yang diinginkan, khususnya ketika digunakan dengan font yang memiliki tinggi x-height atau ketika menggunakan text-decoration-skip-ink.

Ini adalah contoh teks dengan underline posisi otomatis.

Text Underline Position: Under

Ini adalah contoh teks dengan underline posisi di bawah teks.
