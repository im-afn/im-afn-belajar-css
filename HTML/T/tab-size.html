tab-size

tab-size dalam CSS digunakan untuk mengatur lebar karakter tab dalam elemen teks preformatted, seperti pre atau code. Properti ini berguna ketika kamu ingin mengontrol seberapa lebar tab di render dalam konten yang mengandung whitespace.

Tab Size Default (8)

function example() {
    console.log('Hello, World!');
    console.log('This is a tab size of 8.');
}

Tab Size Custom (4)

function example() {
    console.log('Hello, World!');
    console.log('This is a tab size of 4.');
}

Tab Size with Length (2em)

function example() {
    console.log('Hello, World!');
    console.log('This is a tab size of 2em.');
}
