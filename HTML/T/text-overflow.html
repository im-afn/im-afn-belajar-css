text-overflow

text-overflow dalam CSS digunakan untuk menentukan bagaimana konten teks yang melampaui batas elemen harus ditampilkan. Properti ini sering digunakan dalam kombinasi dengan white-space: nowrap dan overflow: hidden untuk memotong atau menyembunyikan teks yang terlalu panjang.

Text Overflow: Clip

Teks ini akan dipotong tanpa tanda-tanda khusus. Teks ini sangat panjang dan melampaui batas elemen.

Text Overflow: Ellipsis

Teks ini akan menampilkan elipsis di akhir. Teks ini sangat panjang dan melampaui batas elemen.

Text Overflow: Custom String

Teks ini akan menampilkan string kustom di akhir. Teks ini sangat panjang dan melampaui batas elemen.
