text-decoration

text-decoration dalam CSS digunakan untuk menambahkan dekorasi ke teks seperti garis bawah (underline), garis di atas (overline), garis tengah (line-through), atau kombinasi dari semuanya. Properti ini juga dapat digunakan untuk mengatur warna dan gaya garis dekorasi.

Text Decoration: None

Teks ini tidak memiliki dekorasi.

Text Decoration: Underline

Teks ini memiliki garis bawah.

Text Decoration: Overline

Teks ini memiliki garis di atas.

Text Decoration: Line-Through

Teks ini memiliki garis tengah.

Text Decoration: Underline and Overline

Teks ini memiliki garis bawah dan garis di atas.

Text Decoration: Custom

Teks ini memiliki garis bawah bergelombang berwarna merah.
