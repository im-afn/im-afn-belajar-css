text-emphasis-style

text-emphasis-style dalam CSS digunakan untuk menentukan gaya simbol penekanan pada teks, terutama dalam konteks bahasa Asia. Properti ini memungkinkan kamu untuk memilih dari beberapa gaya simbol yang digunakan untuk memberikan penekanan.

Text Emphasis Style: Filled

Teks ini memiliki penekanan berisi berwarna merah.

Text Emphasis Style: Open

Teks ini memiliki penekanan terbuka berwarna biru.

Text Emphasis Style: Dot

Teks ini memiliki penekanan berupa titik berwarna hijau.

Text Emphasis Style: Circle

Teks ini memiliki penekanan berupa lingkaran berwarna ungu.

Text Emphasis Style: Square

Teks ini memiliki penekanan berupa kotak berwarna oranye.
