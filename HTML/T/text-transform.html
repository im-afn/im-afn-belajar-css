text-transform

text-transform dalam CSS digunakan untuk mengontrol kapitalisasi teks dalam elemen. Properti ini memungkinkan Anda untuk mengubah teks menjadi huruf kapital, huruf kecil, atau format kapitalisasi lainnya tanpa mengubah konten HTML itu sendiri.

Text Transform: None

Ini adalah contoh teks tanpa transformasi.

Text Transform: Capitalize

ini adalah contoh teks dengan huruf pertama dari setiap kata menjadi huruf kapital.

Text Transform: Uppercase

ini adalah contoh teks dengan semua huruf menjadi huruf kapital.

Text Transform: Lowercase

INI ADALAH CONTOH TEKS DENGAN SEMUA HURUF MENJADI HURUF KECIL.

Text Transform: Full-Width

Ini adalah contoh teks dengan semua karakter diubah ke bentuk full-width. こんにちは。
