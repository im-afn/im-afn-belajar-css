caret-color

Properti caret-color dalam CSS digunakan untuk menentukan warna kursor teks (caret) di dalam elemen input atau teks yang dapat diedit. Properti ini sangat berguna untuk memastikan kursor teks terlihat jelas terhadap latar belakang dan teks di sekitarnya, meningkatkan pengalaman pengguna saat menulis atau mengedit teks.

  • auto: Mengatur warna kursor teks berdasarkan konteks elemen. Biasanya mengikuti warna teks atau latar belakang elemen.
  • color: Nilai warna yang ditentukan secara eksplisit (misalnya, red, #ff0000, rgb(255, 0, 0)).
  • initial: Mengatur properti ke nilai default (biasanya auto)
  • inherit: Mewarisi nilai properti dari elemen induknya.
