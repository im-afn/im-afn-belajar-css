color

Properti color digunakan untuk mengatur warna teks dalam sebuah elemen HTML. Properti ini menerima berbagai jenis nilai yang memungkinkan Anda untuk mengatur warna teks dengan fleksibel.

  1. Kata Kunci (Keyword): Nama warna yang telah didefinisikan.
    • Contoh: red, blue, green, black, white, dll.
  2. Nilai Hexadecimal: Menggunakan triplet heksadesimal.
    • Contoh: #RRGGBB, #RGB
  3. RGB: Menggunakan model warna RGB.
    • Contoh: rgb(255, 0, 0)
  4. RGBA: Menggunakan model warna RGB dengan kanal alfa (transparansi).
    • Contoh: rgba(255, 0, 0, 0.5)
  5. HSL: Menggunakan model warna HSL (hue, saturation, lightness).
    • Contoh: hsl(120, 100%, 50%)
  6. HSLA: Menggunakan model warna HSL dengan kanal alfa.
    • Contoh: hsla(120, 100%, 50%, 0.5)
