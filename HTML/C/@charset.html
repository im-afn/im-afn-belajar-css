@charset

Aturan @charset dalam CSS digunakan untuk menentukan encoding karakter dari stylesheet. Aturan ini harus diletakkan di baris pertama dari file CSS, sebelum ada karakter lain (bahkan komentar) untuk dapat berfungsi dengan baik.

Kompabilitas: Membantu browser mengetahui encoding karakter yang digunakan sehingga dapat menampilkan konten dengan benar.

Internasionalisasi: Penting untuk stylesheet yang mengandung karakter non-ASCII agar teks ditampilkan dengan benar.

Dengan demikian, penggunaan aturan @charset memastikan bahwa browser akan menginterpretasikan karakter dalam file CSS sesuai dengan encoding yang ditentukan, yang penting untuk konsistensi tampilan dan fungsionalitas di berbagai platform dan browser.
