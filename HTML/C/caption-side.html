caption-side

Properti caption-side dalam CSS digunakan untuk menentukan posisi penempatan caption (keterangan) sebuah tabel. Caption dapat ditempatkan di atas atau di bawah tabel, dan properti ini memberikan kontrol atas penempatan tersebut.

  • top: Caption ditempatkan di atas tabel. Ini adalah nilai default.
  • bottom: Caption ditempatkan di bawah tabel.
  • initial: Mengatur properti ke nilai default (top).
  • inherit: Mewarisi properti ini dari elemen induknya.
