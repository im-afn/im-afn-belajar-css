counter-reset

Properti counter-reset digunakan untuk menginisialisasi atau mereset satu atau lebih penghitung (counter) CSS ke nilai tertentu. Penghitung CSS adalah variabel yang dipelihara oleh CSS yang nilainya dapat ditingkatkan atau dikurangi oleh aturan CSS, dan kemudian digunakan untuk pembuatan konten dengan properti content
