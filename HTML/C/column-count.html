column-count

Properti column-count dalam CSS digunakan untuk menentukan jumlah kolom yang diinginkan untuk tata letak multi-kolom pada elemen. Ini memudahkan dalam mengatur teks atau konten lain menjadi beberapa kolom, mirip dengan tata letak kolom surat kabar.
