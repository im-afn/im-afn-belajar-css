counter-set

Properti counter-set di CSS digunakan untuk mengatur nilai penghitung (counter) CSS ke nilai tertentu. Ini berbeda dari counter-reset, yang menginisialisasi atau mereset penghitung. Dengan counter-set, Anda bisa langsung menetapkan nilai tertentu pada penghitung tanpa meresetnya terlebih dahulu.
