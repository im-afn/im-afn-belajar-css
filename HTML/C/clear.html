clear

Properti clear dalam CSS digunakan untuk mengontrol perilaku elemen mengikuti elemen float. Properti ini sering digunakan untuk mencegah elemen bersebelahan dengan elemen yang diatur menggunakan properti float.

Properti clear sering digunakan dalam tata letak halaman web untuk mengelola penempatan elemen setelah elemen float. Ini sangat berguna untuk memastikan elemen berikutnya ditempatkan dengan benar tanpa tumpang tindih atau penempatan yang tidak diinginkan. Properti ini juga sering digunakan bersama dengan elemen "clearfix" untuk

  • none: Default. Elemen tidak akan mempengaruhi elemen float sebelumnya.
  • left: Elemen tidak akan menutupi elemen float di sisi kiri.
  • right: Elemen tidak akan menutupi elemen float di sisi kanan.
  • both: Elemen tidak akan menutupi elemen float di kedua sisi.
  • inline-start: Elemen tidak akan menutupi elemen float di sisi inline-start (tergantung pada arah penulisan teks).
  • inline-end: Elemen tidak akan menutupi elemen float di sisi inline-end (tergantung pada arah penulisan teks).
  • initial: Mengatur properti ke nilai default (none).
  • inherit: Mewarisi properti dari elemen induknya.
