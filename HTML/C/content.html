content

Properti content dalam CSS digunakan untuk menentukan konten yang akan ditampilkan di dalam sebuah elemen menggunakan pseudo-elemen seperti ::before dan ::after. Properti ini sering digunakan untuk menambahkan konten tambahan atau dekoratif ke dalam sebuah elemen.

  • normal: Nilai default. Tidak menambahkan konten apa pun.
  • none: Tidak menampilkan konten apapun.
  • String: Menampilkan konten dalam bentuk teks.
  • URL: Menampilkan konten dalam bentuk gambar.
  • Escape characters: Karakter-karakter tertentu seperti \A (untuk baris baru).

Ini adalah teks di dalam kotak.
