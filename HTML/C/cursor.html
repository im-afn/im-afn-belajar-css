cursor

Properti cursor dalam CSS digunakan untuk menentukan jenis kursor mouse yang akan ditampilkan ketika pengguna mengarahkan mouse mereka ke suatu elemen. Ini membantu memberikan umpan balik visual yang sesuai dengan interaksi yang diharapkan, seperti menunjukkan bahwa suatu elemen dapat diklik atau diseret.

  1. default: Kursor default (biasanya panah).
  2. auto: Kursor default yang dipilih oleh browser.
  3. none: Menyembunyikan kursor.
  4. context-menu: Kursor untuk konteks menu.
  5. help: Kursor tanda tanya, biasanya digunakan untuk bantuan.
  6. pointer: Kursor tangan yang menunjuk, biasanya digunakan untuk tautan.
  7. progress: Kursor menunjukkan operasi sedang berjalan, tetapi pengguna masih dapat berinteraksi.
  8. wait: Kursor menunjukkan operasi sedang berjalan, pengguna tidak dapat berinteraksi.
  9. cell: Kursor menunjukkan sel tabel.
  10. crosshair: Kursor crosshair (tanda silang).
  11. text: Kursor I-bar, digunakan untuk teks yang dapat disorot.
  12. vertical-text: Kursor untuk teks vertikal.
  13. alias: Kursor untuk alias.
  14. copy: Kursor menunjukkan salinan akan dibuat.
  15. move: Kursor untuk memindahkan sesuatu.
  16. no-drop: Kursor menunjukkan bahwa objek tidak dapat dijatuhkan di sini.
  17. not-allowed: Kursor tidak diizinkan.
  18. grab: Kursor tangan terbuka, menunjukkan sesuatu dapat diambil.
  19. grabbing: Kursor tangan tertutup, menunjukkan sesuatu sedang dipegang.
  20. all-scroll: Kursor untuk menggulir semua arah.
  21. col-resize: Kursor untuk mereset ukuran kolom (panah kiri-kanan).
  22. row-resize: Kursor untuk mereset ukuran baris (panah atas-bawah).
  23. n-resize: Kursor untuk mereset ukuran ke utara (atas).
  24. e-resize: Kursor untuk mereset ukuran ke timur (kanan).
  25. s-resize: Kursor untuk mereset ukuran ke selatan (bawah).
  26. w-resize: Kursor untuk mereset ukuran ke barat (kiri).
  27. ne-resize: Kursor untuk mereset ukuran ke timur laut (kanan atas).
  28. nw-resize: Kursor untuk mereset ukuran ke barat laut (kiri atas).
  29. se-resize: Kursor untuk mereset ukuran ke tenggara (kanan bawah).
  30. sw-resize: Kursor untuk mereset ukuran ke barat daya (kiri bawah).
  31. ew-resize: Kursor untuk mereset ukuran kiri-kanan.
  32. ns-resize: Kursor untuk mereset ukuran atas-bawah.
  33. nesw-resize: Kursor untuk mereset ukuran diagonal (timur laut ke barat daya).
  34. nwse-resize: Kursor untuk mereset ukuran diagonal (barat laut ke tenggara).
  35. url(): Menggunakan gambar kursor kustom dari URL yang diberikan.
