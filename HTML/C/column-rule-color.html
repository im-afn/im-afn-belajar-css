column-rule-color

Properti column-rule-color dalam CSS digunakan untuk mengatur warna garis pemisah antara kolom dalam tata letak multi-kolom. Ini adalah bagian dari properti shorthand column-rule yang mencakup lebar (column-rule-width), gaya (column-rule-style), dan warna (column-rule-color).

  • Nama warna (misalnya, red, blue, green)
  • Nilai heksadesimal (misalnya, #ff0000, #00ff00)
  • Nilai RGB (misalnya, rgb(255, 0, 0))
  • Nilai RGBA (misalnya, rgba(255, 0, 0, 0.5))
  • Nilai HSL (misalnya, hsl(120, 100%, 50%))
  • Nilai HSLA (misalnya, hsla(120, 100%, 50%, 0.5))
