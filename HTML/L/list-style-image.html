list-style-image

Properti list-style-image dalam CSS digunakan untuk menentukan gambar yang akan digunakan sebagai penanda (marker) untuk item dalam daftar. Ini memungkinkan Anda untuk mengganti penanda default (seperti bulatan, angka, atau huruf) dengan gambar khusus yang Anda pilih.
