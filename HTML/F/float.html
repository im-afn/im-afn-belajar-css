float

Properti CSS float digunakan untuk mengatur posisi suatu elemen di dalam layout halaman web. Saat menggunakan float, elemen tersebut ditarik ke sisi kiri atau kanan dari elemen-elemen lainnya di sekitarnya, dan elemen-elemen lainnya akan mengelilinginya.

  • left: Mengapungkan elemen ke sisi kiri.
  • right: Mengapungkan elemen ke sisi kanan.
  • none: Default. Elemen tidak mengapung dan mengikuti aliran dokumen normal.
  • inherit: Mengambil nilai float dari elemen induknya.

Contoh Penggunaan Properti Float

float: left; Elemen akan mengapung di sisi kiri dari halaman.
