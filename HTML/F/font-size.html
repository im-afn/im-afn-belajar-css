font-size

Properti font-size dalam CSS digunakan untuk menentukan ukuran teks. Properti ini dapat mengambil berbagai jenis nilai, termasuk panjang absolut, panjang relatif, ukuran berdasarkan kata kunci, dan ukuran berdasarkan persentase.

  • Panjang Absolut: px, pt, cm, mm, in, pc Contoh: 12px, 10pt
  • Panjang Relatif: em, rem, vw, vh, ex, ch Contoh: 1.5em, 2rem
  • Kata Kunci Absolut: xx-small, x-small, small, medium, large, x-large, xx-large Contoh: medium, large
  • Kata Kunci Relatif: larger, smaller Contoh: larger, smaller
  • Persentase: Mengatur ukuran font relatif terhadap elemen induknya. Contoh: 150%

Contoh Penggunaan font-size

Ukuran font ini diatur menggunakan piksel (px).

Ukuran font ini diatur menggunakan em.

Ukuran font ini diatur menggunakan rem.

Ukuran font ini diatur menggunakan persentase.

Ukuran font ini diatur menggunakan kata kunci.
