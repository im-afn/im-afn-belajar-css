@font-feature-value

Aturan @font-feature-values dalam CSS digunakan untuk mendefinisikan dan mengelompokkan pengaturan fitur OpenType yang kemudian dapat diterapkan ke jenis font menggunakan properti font-variant atau font-feature-settings. Aturan ini mempermudah pengelolaan fitur-fitur font yang kompleks dan membuat CSS lebih rapi.
