@font-face

Aturan @font-face dalam CSS digunakan untuk memuat jenis font khusus dari server ke halaman web Anda, sehingga Anda dapat menggunakannya untuk gaya teks di situs Anda tanpa bergantung pada jenis font yang tersedia di perangkat pengguna.

  • font-family: Nama yang diberikan untuk jenis font yang akan digunakan di CSS.
  • src: Lokasi file jenis font yang di-hosting. Ini bisa berupa URL ke file font atau direktori font di server Anda.
  • font-weight (opsional): Berat font yang diinginkan (misalnya, normal atau bold).
  • font-style (opsional): Gaya font yang diinginkan (misalnya, normal atau italic).
