filter

Properti filter dalam CSS digunakan untuk menerapkan efek grafis seperti blur, kecerahan, kontras, dan lainnya pada elemen. Properti ini memungkinkan Anda untuk memanipulasi tampilan elemen tanpa menggunakan alat pengeditan gambar.

  • none: Tidak ada efek (default).
  • blur(px): Menerapkan efek blur pada elemen.
  • brightness(%): Menyesuaikan kecerahan elemen
  • contrast(%): Menyesuaikan kontras elemen.
  • drop-shadow(h-shadow v-shadow blur spread color): Menambahkan bayangan pada elemen.
  • grayscale(%): Menerapkan filter grayscale pada elemen.
  • hue-rotate(deg): Mengatur hue warna elemen.
  • invert(%): Membalikkan warna elemen.
  • opacity(%): Menyesuaikan opacity elemen.
  • saturate(%): Menyesuaikan saturasi elemen.
  • sepia(%): Menerapkan efek sepia pada elemen.
  • url(): Menentukan filter SVG eksternal.
