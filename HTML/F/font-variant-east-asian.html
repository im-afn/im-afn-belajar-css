font-variant-east-asian

Properti font-variant-east-asian dalam CSS digunakan untuk mengontrol tampilan teks Asia Timur, seperti karakter Cina, Jepang, dan Korea. Properti ini memungkinkan Anda untuk menentukan berbagai variasi gaya yang berlaku untuk teks Asia Timur, seperti bentuk tradisional atau bentuk modern, serta bentuk khusus lainnya.

  • normal: Default. Teks ditampilkan tanpa perubahan khusus pada karakter Asia Timur.
  • ruby: Mengaktifkan bentuk anotasi ruby.
  • jis78: Menggunakan glif sesuai standar JIS X 0208-1978.
  • jis83: Menggunakan glif sesuai standar JIS X 0208-1983.
  • jis90: Menggunakan glif sesuai standar JIS X 0213-1990.
  • jis04: Menggunakan glif sesuai standar JIS X 0213-2004.
  • simplified: Menggunakan glif untuk karakter yang disederhanakan.
  • traditional: Menggunakan glif untuk karakter tradisional.
  • full-width: Menggunakan glif dengan lebar penuh.
  • proportional-width: Menggunakan glif dengan lebar proporsional.
  • ruby full-width: Mengaktifkan anotasi ruby dan menggunakan glif dengan lebar penuh.
  • ruby proportional-width: Mengaktifkan anotasi ruby dan menggunakan glif dengan lebar proporsional.
  • inherit: Mengambil nilai dari elemen induk.
  • initial: Mengatur properti ke nilai defaultnya (normal).
  • unset: Menghapus nilai yang diberikan dan mengembalikan properti ke nilai aslinya.
