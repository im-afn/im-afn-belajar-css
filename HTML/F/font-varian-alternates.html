font-variant-alternates

Properti font-variant-alternates dalam CSS digunakan untuk mengontrol berbagai bentuk alternatif dari glif dalam sebuah font. Ini memungkinkan Anda untuk memilih variasi khusus dari glif yang disediakan oleh font OpenType. Nilai-nilai untuk properti ini cukup bervariasi dan bergantung pada font yang digunakan serta fitur yang didukung oleh font tersebut.

  • normal: Default. Teks ditampilkan tanpa alternatif apapun.
  • historical-forms: Menggunakan bentuk-bentuk sejarah dari glif jika tersedia.
  • stylistic(< feature-value-name>): Menggunakan fitur gaya tertentu.
  • styleset(< feature-value-name>): Menggunakan set gaya tertentu.
  • character-variant(< feature-value-name>): Menggunakan varian karakter tertentu.
  • swash(< feature-value-name>): Menggunakan bentuk swash tertentu.
  • ornaments(< feature-value-name>): Menggunakan ornamen tertentu.
  • annotation(< feature-value-name>): Menggunakan bentuk anotasi tertentu.

Contoh Penggunaan font-variant-alternates

Ini adalah teks dengan font-variant-alternates: normal.

Ini adalah teks dengan font-variant-alternates: historical-forms.

Ini adalah teks dengan font-variant-alternates: stylistic(1).
