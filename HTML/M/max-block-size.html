max-block-size

Properti max-block-size dalam CSS digunakan untuk menentukan ukuran maksimum elemen dalam dimensi blok. Ini berarti menetapkan tinggi maksimum dalam mode penulisan horizontal dan lebar maksimum dalam mode penulisan vertikal.
