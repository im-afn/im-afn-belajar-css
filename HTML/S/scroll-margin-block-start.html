scroll-margin-block-start

Properti scroll-margin-block-start dalam CSS digunakan untuk menambahkan margin di sisi awal blok elemen (atas dalam konteks tata letak horizontal) dalam konteks pengguliran. Properti ini berguna untuk memberikan ruang ekstra di atas elemen ketika elemen tersebut digulirkan ke dalam tampilan.
