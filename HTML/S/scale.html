scale

Properti scale dalam CSS digunakan untuk mengubah ukuran elemen dengan memperbesar atau memperkecilnya. Properti ini adalah bagian dari modul CSS Transforms dan memungkinkan kamu untuk memperbesar atau memperkecil elemen pada sumbu X dan Y, serta pada sumbu Z dalam ruang tiga dimensi.
