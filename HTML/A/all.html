all

Properti all dalam CSS adalah properti yang digunakan untuk menerapkan satu set nilai properti CSS ke semua elemen dalam sebuah dokumen. Ini menggabungkan penggunaan seluruh properti CSS dalam satu aturan, memberikan kemampuan untuk mengatur secara global nilai-nilai properti untuk semua elemen di halaman web.

all: initial|inherit|unset;

Contoh 1: Mengatur Ulang Semua Properti CSS:
Misalkan Anda ingin mengatur ulang semua properti CSS untuk semua elemen ke nilai awalnya. Anda dapat menggunakan properti all: initial; untuk melakukan ini.

Contoh 2: Menghapus Properti CSS yang Diturunkan:
Misalkan Anda ingin menghapus semua properti CSS yang diturunkan dari elemen-elemen di dalam suatu area konten, kecuali yang telah ditetapkan secara langsung dalam aturan CSS. Anda dapat menggunakan properti all: unset; untuk melakukan ini.

1. initial
Nilai initial mengatur nilai semua properti CSS kembali ke nilai awalnya atau nilai bawaan yang ditentukan oleh spesifikasi CSS. Dengan kata lain, penggunaan all: initial; akan mengembalikan semua properti CSS untuk semua elemen ke nilai defaultnya. Misalnya, jika Anda memiliki elemen yang memiliki properti color, font-size, dan lainnya yang telah disesuaikan sebelumnya, menggunakan all: initial; akan mengembalikan semua properti tersebut ke nilai defaultnya.

2. inherit
Nilai inherit menyebabkan elemen mewarisi nilai properti dari elemen induknya. Jika nilai properti tidak ditetapkan pada elemen tersebut, maka nilai properti akan diwarisi dari elemen induknya. Misalnya, jika Anda memiliki sebuah elemen yang memiliki properti color yang ditetapkan secara spesifik, menggunakan all: inherit; akan menyebabkan elemen tersebut mewarisi nilai color dari elemen induknya.

3. unset
Nilai unset memberikan efek berbeda tergantung pada apakah properti tersebut diwarisi dari elemen induk atau tidak:
Jika properti tersebut diwarisi dari elemen induk, maka nilai properti akan disetel ke inherit. Jika properti tersebut tidak diwarisi dari elemen induk, maka nilai properti akan disetel ke initial.

Dengan kata lain, penggunaan all: unset; akan menghapus semua properti CSS yang diturunkan dari elemen-elemen di dalam area tertentu, kecuali properti-properti yang telah ditetapkan secara langsung dalam aturan CSS, dan nilai properti yang tidak diwarisi akan dikembalikan ke nilai defaultnya.
