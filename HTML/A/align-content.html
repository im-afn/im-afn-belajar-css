align-content

Properti align-content dalam CSS digunakan untuk mengatur bagaimana ruang di antara dan di sekitar baris dalam container fleksibel (flex container) atau grid container didistribusikan. Properti ini terutama berguna ketika Anda memiliki lebih dari satu baris item dalam flexbox atau grid dan ingin mengontrol penempatan baris tersebut secara vertikal dalam container.

Kapan Menggunakan:
Flexbox: Properti ini berguna ketika Anda memiliki beberapa baris atau kolom item dalam kontainer flex dan ingin mengatur bagaimana ruang ekstra di antara baris atau kolom tersebut didistribusikan.

Grid Layout: Properti ini juga bisa digunakan dalam grid layout untuk mengatur distribusi ruang ekstra di antara baris atau kolom grid.

Dengan menggunakan align-content, Anda dapat mengontrol distribusi ruang ekstra di antara baris atau kolom dalam sebuah kontainer flex atau grid, memberikan fleksibilitas tambahan dalam tata letak responsif dan kompleks.

align-content: stretch|center|flex-start|flex-end|space-between|space-around|space-evenly|initial|inherit;
