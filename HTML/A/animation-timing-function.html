animation-timong-function

Properti animation-timing-function pada CSS digunakan untuk menentukan kurva kecepatan animasi. Kurva kecepatan ini mendefinisikan waktu yang diperlukan oleh animasi untuk berubah dari satu set gaya CSS ke set gaya lainnya. Dengan menggunakan properti ini, perubahan antar gaya dapat terjadi dengan mulus

  1. linear: Animasi memiliki kecepatan yang sama dari awal hingga akhir.
  2. ease: Nilai default. Animasi memiliki awalan lambat, kemudian cepat, sebelum berakhir secara perlahan.
  3. ease-in: Animasi memiliki awalan lambat
  4. ease-out: Animasi memiliki akhiran lambat.
  5. ease-in-out: Animasi memiliki awalan dan akhiran lambat
