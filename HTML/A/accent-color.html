accent-color

Properti accent-color dalam CSS adalah fitur yang relatif baru yang memungkinkan Anda untuk menentukan warna aksen untuk kontrol formulir dan elemen interaktif seperti kotak centang (checkbox), tombol radio, input rentang (range inputs), dan bilah kemajuan (progress bars). Properti ini membantu menjaga konsistensi tema warna di seluruh elemen UI aplikasi web Anda tanpa harus menata setiap kontrol secara manual.

accent-color: auto|color|initial|inherit;
