align-items

Properti align-items dalam CSS digunakan untuk mengatur cara item dalam sebuah kontainer flex atau grid diatur sepanjang sumbu lintang (axis vertikal). Properti ini mengontrol posisi vertikal item dalam sebuah kontainer, mirip dengan properti justify-content yang mengontrol posisi horizontal.

Kapan Menggunakan:
Flexbox: Properti ini sangat berguna ketika Anda ingin mengatur posisi vertikal item-item dalam sebuah kontainer flex.

Grid Layout: Properti ini juga bisa digunakan dalam grid layout untuk mengatur posisi vertikal item dalam baris atau kolom grid.

Dengan menggunakan align-items, Anda dapat mengontrol posisi vertikal item dalam sebuah kontainer flex atau grid, memberikan fleksibilitas tambahan dalam tata letak responsif dan kompleks.

align-items: normal|stretch|positional alignment|flex-start|flex-end|baseline|initial|inherit;
