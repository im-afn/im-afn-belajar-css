animation

Properti animation dalam CSS digunakan untuk membuat animasi pada elemen-elemen HTML. Dengan properti ini, Anda dapat mendefinisikan berbagai aspek dari animasi, seperti durasi, timing function, delay, jumlah pengulangan, dan banyak lagi.

Dengan menggunakan properti animation, Anda dapat membuat animasi yang menarik dan dinamis untuk meningkatkan pengalaman pengguna di situs web Anda.

animation: name duration timing-function delay iteration-count direction fill-mode play-state;





animation-delay: time|initial|inherit;

waktu (time):
Nilai ini menetapkan penundaan dalam detik (s) atau milidetik (ms) sebelum animasi dimulai. Contoh: animation-delay: 2s; atau animation-delay: 500ms;.

initial:
Mengatur properti ke nilai awalnya, yaitu 0s. Ini berarti animasi akan dimulai segera. Contoh: animation-delay: initial;.

inherit:
Mewarisi nilai dari elemen induknya. Contoh: animation-delay: inherit;.
