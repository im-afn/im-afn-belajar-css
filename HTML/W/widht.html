widht

width dalam CSS digunakan untuk menentukan lebar elemen. Properti ini dapat diterapkan pada berbagai jenis elemen, termasuk elemen blok, elemen inline, elemen inline-block, dan elemen lainnya. Properti width mengontrol lebar konten elemen, tidak termasuk padding, border, atau margin.
