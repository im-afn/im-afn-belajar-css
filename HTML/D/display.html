display

Properti display dalam CSS digunakan untuk menentukan bagaimana sebuah elemen HTML harus ditampilkan di halaman web. Properti ini memengaruhi tata letak elemen dan perilaku box model dari elemen tersebut.

Nilai-nilai yang Tersedia

  1. block: Elemen ditampilkan sebagai elemen blok, yang memulai pada baris baru dan mengambil lebar penuh yang tersedia.
  2. inline: Elemen ditampilkan sebagai elemen sebaris, yang tidak memulai baris baru dan hanya mengambil lebar yang diperlukan.
  3. inline-block: Elemen ditampilkan sebagai elemen sebaris tetapi dapat memiliki lebar dan tinggi yang diatur.
  4. flex: Elemen ditampilkan sebagai wadah fleksibel, menggunakan Flexbox.
  5. inline-flex: Elemen ditampilkan sebagai wadah fleksibel sebaris.
  6. grid: Elemen ditampilkan sebagai wadah grid, menggunakan CSS Grid Layout.
  7. inline-grid: Elemen ditampilkan sebagai wadah grid sebaris.
  8. table: Elemen ditampilkan sebagai tabel (block-level table).
  9. inline-table: Elemen ditampilkan sebagai tabel sebaris.
  10. table-row: Elemen ditampilkan sebagai baris tabel.
  11. table-cell: Elemen ditampilkan sebagai sel tabel.
  12. table-column: Elemen ditampilkan sebagai kolom tabel.
  13. table-column-group: Elemen ditampilkan sebagai grup kolom tabel.
  14. table-row-group: Elemen ditampilkan sebagai grup baris tabel.
  15. table-header-group: Elemen ditampilkan sebagai grup header tabel.
  16. table-footer-group: Elemen ditampilkan sebagai grup footer tabel.
  17. list-item: Elemen ditampilkan sebagai item daftar.
  18. run-in: Elemen ditampilkan sebagai elemen sebaris atau blok tergantung pada konteksnya.
  19. none: Elemen tidak ditampilkan.
  20. contents: Elemen dihapus, tetapi anak-anaknya tetap ditampilkan.
  21. initial: Mengatur properti ke nilai defaultnya.
  22. inherit: Mengatur properti agar mewarisi nilai dari elemen induknya.
