border-block-start-color

Properti CSS border-block-start-color digunakan untuk menentukan warna border (garis tepi) pada sisi awal blok elemen, berdasarkan arah penulisan teks. Ini memungkinkan Anda untuk mengatur warna border awal blok dalam satu deklarasi.

  • Nama Warna: e.g., red, blue, green
  • Nilai Heksadesimal: e.g., #ff0000, #00ff00
  • Nilai RGB: e.g., rgb(255, 0, 0), rgb(0, 255, 0)
  • Nilai RGBA: e.g., rgba(255, 0, 0, 0.5), rgba(0, 255, 0, 0.5)
