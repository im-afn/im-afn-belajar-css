border-inline-style

Properti CSS border-inline-style digunakan untuk menetapkan gaya border pada sisi inline dari sebuah elemen. Ini menyatukan semua properti gaya border (seperti border-inline-start-style, border-inline-end-style, border-block-start-style, dan border-block-end-style) dalam satu deklarasi.

Contoh 1: solid
Contoh 2: dashed
Contoh 3: dotted
Contoh 4: double
Contoh 5: groove
Contoh 6: ridge
Contoh 7: inset
Contoh 8: outset
