border-style

Properti CSS border-style digunakan untuk menetapkan gaya border dari elemen. Properti ini bisa mengatur gaya border untuk semua sisi elemen atau untuk setiap sisi secara individual. Nilai yang bisa digunakan meliputi:

  • none
  • hidden
  • dotted
  • dashed
  • solid
  • double
  • groove
  • ridge
  • inset
  • outset
Contoh 1: none
Contoh 2: dotted
Contoh 3: dashed
Contoh 4: solid
Contoh 5: double
Contoh 6: groove
Contoh 7: ridge
Contoh 8: inset
Contoh 9: outset
