box-sizing

Properti box-sizing dalam CSS digunakan untuk menentukan bagaimana total ukuran elemen (width dan height) dihitung. Ini membantu mengontrol apakah padding dan border termasuk dalam ukuran elemen atau tidak.

  • content-box (nilai default)
  • border-box
    content-box:
  1. Ini adalah nilai default.
  2. Width dan height hanya berlaku untuk konten elemen. Padding dan border ditambahkan di luar ukuran yang ditentukan.
  3. Contoh: Jika Anda menetapkan width: 100px dan elemen memiliki padding: 10px dan border: 5px, total lebar elemen akan menjadi 100px + 10px (padding kiri) + 10px (padding kanan) + 5px (border kiri) + 5px (border kanan) = 130px.
    border-box:
  1. Width dan height mencakup konten, padding, dan border.
  2. Contoh: Jika Anda menetapkan width: 100px dan elemen memiliki padding: 10px dan border: 5px, total lebar elemen tetap 100px karena padding dan border dihitung dalam ukuran total.
content-box
Total width: 200px + padding + border
border-box
Total width: 200px (includes padding and border)
