background-repeat

Properti background-repeat dalam CSS digunakan untuk mengontrol apakah dan bagaimana gambar latar belakang diulang di dalam elemen. Properti ini memungkinkan Anda menentukan apakah gambar latar belakang akan diulang secara horizontal, vertikal, atau tidak diulang sama sekali.

  1. repeat: Gambar latar belakang diulang baik secara horizontal maupun vertikal (default).
  2. repeat-x: Gambar latar belakang hanya diulang secara horizontal.
  3. repeat-y: Gambar latar belakang hanya diulang secara vertikal.
  4. no-repeat: Gambar latar belakang tidak diulang sama sekali.
