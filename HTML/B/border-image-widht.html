border-image-widht

Properti CSS border-image-width digunakan untuk menentukan lebar area border-image. Lebar ini dapat ditentukan dalam satuan panjang (misalnya, piksel), dalam persentase, atau sebagai nilai relatif terhadap lebar border asli.

Contoh 1: Width 10px
Contoh 2: Width 20%
Contoh 3: Width 0.5
Contoh 4: Width 10px 20px 30px 40px
