border-inline-end-style

Pengertian Properti CSS border-inline-end-style digunakan untuk menetapkan gaya border pada sisi inline-end dari sebuah elemen. Sisi inline-end mengacu pada sisi akhir dari elemen berdasarkan arah penulisan (writing mode) dari teks dalam elemen tersebut. Dalam mode penulisan LTR (left-to-right), sisi inline-end adalah sisi kanan, sedangkan dalam mode penulisan RTL (right-to-left), sisi inline-end adalah sisi kiri.

Contoh 1: solid
Contoh 2: dashed
Contoh 3: dotted
Contoh 4: double
Contoh 5: groove
Contoh 6: ridge
Contoh 7: inset
Contoh 8: outset
