border-inline-end

Properti CSS border-inline-end digunakan untuk menetapkan semua properti border pada sisi inline-end dari sebuah elemen. Sisi inline-end adalah sisi kanan dalam mode penulisan LTR (left-to-right) dan sisi kiri dalam mode penulisan RTL (right-to-left).

Contoh 1: 5px solid red
Contoh 2: 10px dashed
Contoh 3: 3px solid green
Contoh 4: 2px dashed blue
Contoh 5: 4px dotted purple
