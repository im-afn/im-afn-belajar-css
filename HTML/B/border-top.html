border-top

Properti CSS border-top digunakan untuk menetapkan semua properti border (lebar, gaya, dan warna) untuk sisi atas elemen. Properti ini adalah shorthand untuk mengatur border-top-width, border-top-style, dan border-top-color.

Contoh 1: solid black
Contoh 2: dashed red
Contoh 3: double green
Contoh 4: groove blue
Contoh 5: ridge purple
