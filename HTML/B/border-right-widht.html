border-right-widht

Properti CSS border-right-width digunakan untuk menetapkan lebar border pada sisi kanan elemen. Properti ini dapat menggunakan beberapa nilai yang telah ditentukan seperti thin, medium, thick, atau nilai panjang khusus seperti px, em, dll.

Contoh 1: thin
Contoh 2: medium
Contoh 3: thick
Contoh 4: 2px
Contoh 5: 4px
Contoh 6: 6px
