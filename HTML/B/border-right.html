border-right

Properti CSS border-right digunakan untuk menetapkan semua pengaturan border pada sisi kanan elemen sekaligus, termasuk lebar, gaya, dan warna border.

Contoh 1: 5px solid black
Contoh 2: 10px dashed red
Contoh 3: 15px dotted blue
Contoh 4: 5px double green
Contoh 5: 7px groove purple
Contoh 6: 3px ridge orange
