border-image-repeat

Properti CSS border-image-repeat digunakan untuk menentukan bagaimana gambar yang digunakan dalam border diulang (di-tile) atau diregangkan di sepanjang border elemen.

  • stretch: Menarik gambar untuk mengisi area border.
  • repeat: Mengulang gambar untuk mengisi area border.
  • round: Mengulang gambar, tetapi jika tidak cocok dengan sempurna, gambar akan disesuaikan ukurannya (skala) agar sesuai.
  • space: Mengulang gambar dan menyisakan ruang kosong antara pengulangan jika perlu.
