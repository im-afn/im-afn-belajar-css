border-block-end-style

Properti border-block-end-style dalam CSS digunakan untuk mengatur gaya border (garis tepi) pada sisi akhir dari elemen blok, sesuai dengan arah penulisan teks. Properti ini memungkinkan Anda untuk menentukan bagaimana bentuk border tersebut, seperti solid (padat), dashed (putus-putus), atau dotted (titik-titik).

  • none: Tidak ada border.
  • solid: Border padat.
  • dashed: Border putus-putus.
  • dotted: Border titik-titik.
  • double: Border ganda.
  • groove: Border dengan efek ukiran
  • ridge: Border dengan efek tonjolan.
  • inset: Border dengan efek masuk ke dalam.
  • outset: Border dengan efek keluar.
Contoh border-block-end-style: none
