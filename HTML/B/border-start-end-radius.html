border-start-end-radius

Properti CSS border-start-end-radius digunakan untuk menetapkan radius sudut di bagian awal dari sisi akhir elemen. Dalam penulisan kiri-ke-kanan (LTR), ini biasanya merujuk pada sudut kanan bawah. Properti ini sangat berguna untuk mengatur sudut pada elemen blok dengan arah teks tertentu.

Contoh 1: 10px
Contoh 2: 20px
Contoh 3: 30px
Contoh 4: 40px
