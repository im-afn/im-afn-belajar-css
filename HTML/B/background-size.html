background-size

Properti background-size dalam CSS digunakan untuk mengontrol ukuran gambar latar belakang dalam sebuah elemen. Properti ini memungkinkan Anda menyesuaikan ukuran gambar latar belakang agar sesuai dengan kebutuhan tata letak halaman web Anda.

  1. auto: Gambar latar belakang akan ditampilkan dalam ukuran aslinya (default).
  2. cover: Gambar latar belakang akan diubah ukurannya agar menutupi seluruh area elemen.
  3. contain: Gambar latar belakang akan diubah ukurannya agar sesuai ke dalam area elemen tanpa merubah aspek rasionya.
  4. Nilai numerik: Anda dapat menentukan ukuran gambar dalam piksel (px), persen (%), fraksi (fr), atau kata kunci lainnya.
