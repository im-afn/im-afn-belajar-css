border-inline-start-style

border-inline-start-style adalah properti CSS yang digunakan untuk menentukan gaya garis untuk sisi awal inline dari sebuah elemen. Sisi awal inline akan berbeda tergantung pada arah penulisan teks (LTR atau RTL).

Ini adalah contoh border dengan border-inline-start-style: none;

hidden, itu biasanya berarti properti tersebut ada, tetapi tidak terlihat oleh pengguna. hidden;, elemen tetap memperoleh ruang di layout, tetapi tidak terlihat oleh pengguna.

Ini adalah contoh border dengan border-inline-start-style: hidden;
