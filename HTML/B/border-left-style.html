border-inline-style

Properti CSS border-left-style digunakan untuk menetapkan gaya border pada sisi kiri dari sebuah elemen. Ini memungkinkan pengaturan jenis garis border (misalnya solid, dashed, dotted) pada sisi kiri elemen.

Contoh 1: solid
Contoh 2: dashed
Contoh 3: dotted
Contoh 4: double
Contoh 5: groove
Contoh 6: ridge
Contoh 7: inset
Contoh 8: outset
