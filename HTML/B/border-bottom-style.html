border-bottom-style

Properti CSS border-bottom-style digunakan untuk menentukan gaya dari border bagian bawah suatu elemen. Properti ini memungkinkan Anda untuk mengatur apakah border bagian bawah akan memiliki gaya garis lurus, putus-putus, titik-titik, dan sebagainya.

Contoh 1: Border bawah dengan gaya solid
Contoh 2: Border bawah dengan gaya dotted
Contoh 3: Border bawah dengan gaya dashed
Contoh 4: Border bawah dengan gaya double
