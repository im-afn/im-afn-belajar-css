border-inlien-end-color

Properti CSS border-inline-end-color digunakan untuk menetapkan warna border pada sisi inline-end dari elemen. Sisi inline-end mengacu pada sisi akhir dari elemen berdasarkan arah penulisan (writing mode) dari teks dalam elemen tersebut. Dalam mode penulisan LTR (left-to-right), sisi inline-end adalah sisi kanan, sedangkan dalam mode penulisan RTL (right-to-left), sisi inline-end adalah sisi kiri.

Contoh 1: border-inline-end-color merah
Contoh 2: border-inline-end-color biru
Contoh 3: border-inline-end-color #ff00ff
Contoh 4: border-inline-end-color rgb(0, 255, 0)
Contoh 5: border-inline-end-color hsl(240, 100%, 50%)
