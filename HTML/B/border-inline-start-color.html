border-inline-start-color

Properti CSS border-inline-start-color digunakan untuk menetapkan warna border pada sisi inline-start dari sebuah elemen. Sisi inline-start mengacu pada sisi awal dari elemen berdasarkan arah penulisan (writing mode) dari teks dalam elemen tersebut. Dalam mode penulisan LTR (left-to-right), sisi inline-start adalah sisi kiri, sedangkan dalam mode penulisan RTL (right-to-left), sisi inline-start adalah sisi kanan.

Contoh 1: border-inline-start-color merah
Contoh 2: border-inline-start-color biru
Contoh 3: border-inline-start-color hijau
Contoh 4: border-inline-start-color oranye
Contoh 5: border-inline-start-color ungu
Contoh 6: border-inline-start-color hitam
