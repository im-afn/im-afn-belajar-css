inset-block

Properti inset-block dalam CSS adalah bagian dari CSS Logical Properties and Values. Properti ini adalah shorthand untuk mengatur properti inset-block-start dan inset-block-end secara bersamaan. Properti ini menentukan jarak antara elemen dan tepi blok awal serta akhir dari elemen induk. Nilai dari properti ini bergantung pada arah penulisan (writing mode) dari teks.
