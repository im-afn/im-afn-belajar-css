inset-inline-start

Properti inset-inline-start dalam CSS adalah bagian dari CSS Logical Properties and Values. Properti ini menentukan jarak antara tepi inline awal dari sebuah elemen dan elemen induknya. Arah inline tergantung pada arah penulisan (writing mode) dari teks, misalnya, dalam penulisan dari kiri ke kanan (LTR), inline-start akan berada di sisi kiri elemen.
