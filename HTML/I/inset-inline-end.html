inset-inline-end

Properti inset-inline-end dalam CSS adalah bagian dari CSS Logical Properties and Values. Properti ini menentukan jarak antara tepi inline akhir dari sebuah elemen dan elemen induknya. Arah inline tergantung pada arah penulisan (writing mode) dari teks.
