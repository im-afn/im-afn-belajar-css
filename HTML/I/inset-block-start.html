inset-block-start

Properti inset-block-start dalam CSS adalah bagian dari CSS Logical Properties and Values, yang memungkinkan Anda untuk menentukan jarak antara tepi elemen blok awal (start) dan elemen induknya. Nilai dari properti ini bergantung pada arah penulisan (writing mode) dari teks. Dalam penulisan horizontal, inset-block-start biasanya setara dengan top. Dalam penulisan vertikal, itu akan menjadi left atau right, tergantung pada arah penulisan.
