@import

Aturan @import dalam CSS digunakan untuk mengimpor stylesheet eksternal atau internal ke dalam stylesheet saat ini. Aturan ini memungkinkan Anda untuk memisahkan dan mengatur file CSS Anda dengan lebih baik, terutama ketika proyek Anda mulai tumbuh lebih besar dan lebih kompleks.
