inset-block-end

Properti inset-block-end dalam CSS adalah bagian dari CSS Logical Properties and Values. Properti ini digunakan untuk menentukan jarak antara tepi elemen blok akhir (end) dan elemen induknya. Nilai dari properti ini bergantung pada arah penulisan (writing mode) dari teks. Dalam penulisan horizontal, inset-block-end biasanya sama dengan bottom. Dalam penulisan vertikal, itu akan menjadi left atau right, tergantung pada arah penulisan.
