justify-items

Properti justify-items dalam CSS digunakan untuk menyelaraskan item dalam wadah grid atau flex pada sumbu inline (horizontal dalam penulisan LTR, vertikal dalam penulisan RTL). Properti ini menentukan bagaimana item di dalam wadah grid atau flex seharusnya didistribusikan atau diselaraskan sepanjang sumbu inline. Ini berfungsi untuk tata letak grid dan juga bisa digunakan dalam konteks flexbox, terutama ketika flexbox dalam mode kolom.

  • start: Item disejajarkan ke awal wadah.
  • end: Item disejajarkan ke akhir wadah.
  • center: Item disejajarkan ke tengah wadah.
  • stretch: Item diperluas untuk mengisi seluruh wadah (nilai default).
