grid-template-rows

Properti grid-template-rows dalam CSS digunakan untuk mendefinisikan jumlah dan ukuran baris dalam sebuah grid container. Ini memungkinkan Anda mengatur tinggi masing-masing baris dalam grid menggunakan berbagai jenis nilai seperti panjang absolut, fraksi, persentase, dan lainnya.
