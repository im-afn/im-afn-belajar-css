grid

Properti grid dalam CSS adalah shorthand yang dapat digunakan untuk menetapkan semua properti grid dalam satu deklarasi. Ini termasuk pengaturan untuk grid-template-rows, grid-template-columns, grid-template-areas, grid-auto-rows, grid-auto-columns, dan grid-auto-flow.
